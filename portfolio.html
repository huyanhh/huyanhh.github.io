iXercise

This project was a class project for an HCI class. Essentially we had to go out and speak with doctors to get a concrete set of requirements to build a web application for an exercise game that supports the rehabilitation of recovering leukemia patients. Along with my team, I conducted interviews and figured out what to build, then tested the prototype designs with the stakeholders first. Once we learned the exact details of a successful product we built the application. I was in charge of writing to the database and making sure the data meant to be sent through GET/POST requests were integrity checked with MongoDB. Click on the screen to enlarge it.

Skills used: MongoDB, Node.js, D3
